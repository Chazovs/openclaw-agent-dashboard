# PR: Добавление аналитической панели в OpenClaw Dashboard

## 📋 Описание изменений

Добавлена новая аналитическая панель, предоставляющая расширенную аналитику активности агентов, тренды использования токенов и системные метрики в реальном времени.

## 🎯 Цель

Улучшить мониторинг и анализ работы OpenClaw агентов через:
1. Визуализацию трендов активности
2. Мониторинг использования ресурсов
3. Своевременное обнаружение проблем
4. Улучшение UX через интерактивные графики

## ✨ Новые возможности

### 📈 Интерактивные графики
- **График активности агентов** (24ч/7д/30д)
- **График использования токенов** (общее/среднее/топ-5)

### ⚡ Сводная информация
- Карточки статуса системы в реальном времени
- Тренды изменений за последний час
- Автоматическое обновление каждые 30 секунд

### ⚠️ Система уведомлений
- Автоматическое обнаружение проблем
- Критические уведомления (CPU/память)
- Уведомления об ошибках агентов

### 💾 Системные метрики
- Мониторинг CPU и памяти
- Отслеживание времени работы
- Детализированная статистика

## 🔧 Установка и настройка

### Требования
- OpenClaw Dashboard v1.0+
- Node.js 14+
- Chart.js (подключается через CDN)

### Установка
1. Скопируйте новые файлы в проект
2. Обновите `server.js` с новыми импортами
3. Перезапустите сервер

### Конфигурация
Настройки хранятся в `analytics-module.js`:
- Период хранения данных
- Пороги уведомлений
- Пути к данным
