# История активности агентов - Документация

## Обзор

Добавлена новая функциональность для отслеживания истории активности агентов в реальном времени. Теперь можно просматривать временную шкалу активности каждого агента, показывающую когда и какие задачи выполнялись.

## Новые возможности

### 1. Временная шкала активности
- Просмотр истории активности каждого агента
- Визуализация статусов во времени (работает, бездействует, ошибка)
- Отображение выполняемых задач
- Продолжительность каждой активности

### 2. API endpoints

#### GET `/api/agents/:id/activity`
Получение истории активности конкретного агента.

**Параметры:**
- `limit` (опционально) - количество записей (по умолчанию 20)

**Ответ:**
```json
{
  "success": true,
  "agentId": "main_claude",
  "activity": [
    {
      "timestamp": "2026-04-18T04:06:40.055Z",
      "status": "working",
      "task": "Анализ дашборда",
      "duration": 1200000
    }
  ],
  "total": 1
}
```

#### PUT `/api/agents/:id/status` (обновлен)
Теперь поддерживает параметр `task` для записи в историю.

**Тело запроса:**
```json
{
  "status": "working",
  "task": "Выполнение конкретной задачи"
}
```

### 3. WebSocket события

#### `get-agent-activity`
Запрос истории активности агента.

#### `agent-activity`
Ответ с историей активности.

### 4. Пользовательский интерфейс

#### Кнопка "Показать историю активности"
Добавлена на каждую карточку агента. При нажатии:
- Открывается временная шкала активности
- Загружается история агента
- Отображается в хронологическом порядке

#### Временная шкала
- Визуальная временная шкала с точками активности
- Цветовое кодирование статусов:
  - 🟩 Зеленый - работает
  - 🟨 Желтый - бездействует  
  - 🟥 Красный - ошибка
- Детали каждой активности:
  - Время начала
  - Статус
  - Задача
  - Продолжительность

## Техническая реализация

### Серверная часть (`server-simple.js`)

#### Новые структуры данных
```javascript
// История активности агентов
let agentActivityHistory = {
  'agent_id': [
    {
      timestamp: ISOString,
      status: 'working' | 'idle' | 'error',
      task: string,
      duration: number // в миллисекундах
    }
  ]
};
```

#### Новые функции
1. `addActivityRecord(agentId, activity)` - добавление записи активности
2. `getAgentActivityHistory(agentId, limit)` - получение истории
3. `getRandomTask(status)` - генерация демо-задач

#### Логика обновления истории
- При каждом изменении статуса агента создается новая запись
- Автоматически вычисляется продолжительность предыдущей активности
- Ограничение истории последними 50 записями на агента

### Клиентская часть (`public/index.html`)

#### Новые CSS классы
- `.timeline-container` - контейнер временной шкалы
- `.timeline-item` - элемент временной шкалы
- `.timeline-status` - индикатор статуса
- `.view-activity-btn` - кнопка просмотра активности

#### Новые JavaScript функции
1. `viewAgentActivity(agentId, agentName)` - открытие временной шкалы
2. `renderTimeline(activities)` - рендеринг временной шкалы
3. `formatDuration(ms)` - форматирование продолжительности
4. `requestAgentActivity(agentId)` - запрос истории через WebSocket

## Использование

### Для разработчиков
```javascript
// Получение истории активности
fetch('/api/agents/main_claude/activity?limit=10')
  .then(response => response.json())
  .then(data => console.log(data.activity));

// Обновление статуса с записью задачи
fetch('/api/agents/main_claude/status', {
  method: 'PUT',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    status: 'working',
    task: 'Конкретная задача'
  })
});
```

### Для пользователей
1. Откройте дашборд по адресу `http://localhost:3001`
2. Найдите карточку нужного агента
3. Нажмите кнопку "📊 Показать историю активности"
4. Просматривайте временную шкалу активности
